# IRO (Issues Reporting Outlet)

## Project Description

IRO (Issues Reporting Outlet) is a simple web-based application developed for CSC426. It enables AAUA students and Akungba community indigenes to report issues affecting their environment, such as security, electricity, road, water, accommodation, and other community-related problems.

## Features

- Report community issues
- User-friendly interface
- Input validation
- Success and error messages
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
